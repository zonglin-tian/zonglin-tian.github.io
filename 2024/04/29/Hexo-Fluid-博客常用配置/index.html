自定义页添加目录

对于自定义页的 .md 文件自动添加目录, 添加步骤:

  1. 全局安装 mardown-toc 包: npm install -g markdown-toc
  2. 在需要添加目录的文件的对应位置使用代码: <!-- toc -->
  3. 在终端输入命令: mardown-toc -i file_name.md
