JSON

JSON (JavaScript Object Notation): 轻量级数据交换格式, 具有简洁, 易读的文本结构 (.json)

基本结构

  • 键值对: 数据以 key: value的形式存储, 键必须是双引号包裹的字符串
  • 值的数据类型
    • 字符串: "name": "Temm"
    • 数字: "Age": 3
    • 布尔值: "is_student": true
    • 空值: "address": null
    • 对象 (嵌套): "university": {"name": "BNU"}
    • 数组: "scores": [90, 85, 77]

语法规则

  • 根元素必须是对象 {} 或数组 []
  • 引号必须用双引号: 单引号或无引号无效
  • 逗号分隔: 键值对之间用逗号分隔, 但末尾不能有逗号
  • 无注释: 不支持标准注释 //, /**/
  • 在线校验: JSONLint

衍生格式

JSONC (JSON with Comments)

  • 扩展名: .jsonc
  • 特点:
    • 支持 // 单行注释和 /**/ 多行注释
    • 保留 JSON 的严格数据结构, 仅增加注释功能

JSON5 (更人性化的 JSON)

  • 扩展名: .json5
  • 特点:
    • 支持注释, 尾随逗号, 单引号字符串, 无引号键名, 十六进制数字等
    • 目标是更易手写和维护
    • 官方库

JSONL (JSON Lines)

  • 扩展名: .jsonl
  • 特点:
    • 每行是一个独立的 JSON 对象, 适合流式处理或日志文件
    • 无需根数组或对象包裹
